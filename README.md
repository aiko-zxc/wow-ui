## RU

### Blizzard Edit Mode

1. Escape
2. Edit Mode
3. Layout
4. Import
5. Копируем [Blizzard UI Layout](blizz-layout/blizz.txt) и применяем

### Platynator

1. Введите команду `/platy`
2. Выберите вкладку **General** → **Import**
3. Копируем [профиль](midnight/platynator.txt)
4. Вставьте в окно импорта и примените профиль

## EN

### Blizzard Edit Mode

1. Escape
2. Edit Mode
3. Layout
4. Import
5. Copy [Blizzard UI Layout](blizz-layout/blizz.txt), paste, and apply

### Platynator

1. Open the game and type `/platy`
2. Go to **General** → **Import**
3. Copy [profile](midnight/platynator.txt)
4. Paste it in the import window and apply
